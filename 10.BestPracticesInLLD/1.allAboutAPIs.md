### Naming best practices
- GET /api/v1/users/123
- use plural, nouns. Avoid verbs in path. Use HTTP verbs to describe the action
- /cancelOrder/123 this is wrong as we are using verb instead PATCH /api/v1/order/123 maybe accept some type in the body
- POST /user -> create a user


### Throttling vs Rate limiting 
- rate limiting where you allow certain number of request to go through
- throttling where i would slow down the responses or drop the request when the resources are under high load
